Python & R packages

The easiest way to use sTiles, no manual download. Each fetches the right libstiles binary automatically on first use.

Python (PyPI)
pip install sTiles
R (GitHub)
remotes::install_github(
  "esmail-abdulfattah/sTiles",
  subdir = "R/sTiles")
